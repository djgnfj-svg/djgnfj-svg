### About Me
현재 한국로봇융합연구원에서 **로봇 관제 대시보드**를 풀스택으로 기획·개발·운영하고 있습니다. TypeScript와 FastAPI를 주로 씁니다.

AI를 실무와 사이드 프로젝트에 적극적으로 붙여보고 있습니다.
- 이력서·채용공고를 분석해 **AI 음성 면접 코치**(VoicePrep)를 멀티에이전트로 설계·운영
- 에빙하우스 망각곡선 기반 **AI 복습 자동화 플랫폼**(Resee) 1인 개발
- LangGraph 멀티에이전트가 **미국 빅테크를 분석해 뉴스레터를 발송**하는 서비스 운영(MorningBrief)
- 채용공고에 맞춰 이력서를 자동 선별·생성하는 하네스, 남은 토큰으로 스스로 실험하는 자율 루프 등 Claude Code 기반 자동화 실험
